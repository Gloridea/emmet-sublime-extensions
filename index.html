emmet-sublime-extensions

Emmet-sublime extensions

Features

data- : @

Provides abbr. for HTML5 data-attributes.

div[@role=page]

... wil produce

<div data-role="page"></div>

Install

  • Make Emmet extensions directory and copy js file into them. ( please refer https://github.com/sergeche/emmet-sublime#extensions-support )
  • Configure Emmet.sublime-settings to define extensions directory.
  • Restart Sublime Text.
